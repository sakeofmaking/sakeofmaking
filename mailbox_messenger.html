Mailbox Messenger

November 19th, 2015

The Mailbox Messenger is a mailbox that texts the appropriate roommate when their mail arrives. Let's say you and your roommate share an apartment mailbox. Retrieving the mail is usually not a joint venture so one of you probably gets their mail, leaving the other roommates mail on the counter only for it to be discovered days later. No more! With the Mailbox Messenger, whoever retrieves the mail can leave their roommates mail in the Mailbox Messenger, set the roommate flag, and close the lid. The Mailbox Messenger will then alert the indicated roommate via text.

This project was made possible by MakerBee's instructable. We'll be borrowing heavily from their designs.

Step 1: Materials + Tools

Materials:

  • (x1) Arduino Yun from adafruit
  • (x1) Temboo account
  • (x1) Twilio account
  • (x1) 9V, 250mA or more, 2.1mm center pin positive adapter or alternative Arduino power supply
  • (x1) breadboard
  • (x2) 10kOhm (brown black orange gold) resistor x2
  • (x1) 5mm red LED
  • (x2) SPST momentary ON pushbutton switch
  • (x5) breadboarding wire
  • (x1) mailbox

Tools:

  • Arduino IDE from Arduino

Step 6: Upload the Code

Connect your Arduino Yun to your computer via wifi or cable. Copy and paste the linked code into your Arduino IDE. Enter your specific Temboo Account, Temboo Application, Temboo Key, Twilio Account SID, Twilio Auth Token, Twilio number, and personal number in the appropriate define statements. Run Verify and then Upload. Give it a test run to make sure everything is working nicely. Adjust the waitTime and codeTime variables if necessary.

Step 7: Install by the Door

I taped a small wooden project box next the door to act as a shelf for the system. I also used some cork to help level the door sensor. This is probably the easiest setup. You might want to try something else. Just remember, if you're powering your Arduino via a wall charger, to place the system within range of a wall outlet.

Step 8: Enjoy

Plug in your system, arm it, close the door, and invite a thief over.
